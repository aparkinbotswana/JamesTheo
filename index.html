About

Projects

Virtual Auslan Interpreter - Javascript/Leap Motion
The Auslan Interpreter is designed to help those who are unable to understand the language, communicate with those who can. Currently, it recognises half the English alphabet. The browser app uses the Leap Motion to interpret hand gestures from the user. Based on these gestures, the program recognises which letter is being signed and displays, as well as speaks the recognised letter.



Gooder Reads - Ruby on Rails
A book review website designed to have readers, rather than critics, write reviews on books they have read, as well as comment on the books other people have read.



Fråga - Javascript/Ruby on Rails
Social app with a focus on geolocation. Users can post any question they like relevant to a particular location. Other users within the vicinity can see and respond to the question.



Tic Tac Toe - Javascript
This project represents my first attempt at creating a website from start to finish, using html, css, javascript and JQuery. Additionally, I programmed a near unbeatable computer opponent for the game. There is only one route a player can take to beat the computer.
